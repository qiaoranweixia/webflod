# 🔒 密码保护 - 完整功能指南

## 🎯 功能总览

密码保护功能允许你为私密视频设置密码，防止未经授权的访问。

### 核心功能

| 功能 | 说明 | 状态 |
|------|------|------|
| 🔐 密码加密 | SHA256 哈希存储 | ✅ |
| 💡 密码提示 | 设置密码提示帮助记忆 | ✅ |
| 📜 访问日志 | 记录每次访问尝试 | ✅ |
| 📊 密码统计 | 成功/失败次数统计 | ✅ |
| 📦 批量操作 | 批量设置/移除密码 | ✅ |
| 🔒 播放验证 | 播放前验证密码 | ✅ |
| 📱 模态框 | 友好的密码输入界面 | ✅ |
| 🛡️ 安全保护 | 错误密码 401 响应 | ✅ |

## 🚀 使用指南

### 设置密码（单个视频）

**方法 1: 从视频设置**
1. 找到想要保护的视频
2. 点击"⚙️ 设置"按钮
3. 在"密码保护"区域输入密码
4. （可选）输入密码提示
5. 点击"应用"
6. ✅ 视频已保护！

**方法 2: 批量设置**
1. 勾选多个视频（点击复选框）
2. 底部出现批量操作栏
3. 点击"🔒 批量加密"
4. 输入密码和提示
5. ✅ 批量设置完成！

### 移除密码

**单个视频:**
1. 点击视频"⚙️ 设置"
2. 密码输入框留空
3. 点击"应用"
4. ✅ 密码已移除

**批量移除:**
1. 勾选多个视频
2. 点击"🔓 批量解密"
3. 确认移除
4. ✅ 批量移除完成

### 观看受保护视频

1. 点击受保护的视频（显示 🔒 标记）
2. 弹出密码输入框
3. 输入正确密码
4. ✅ 开始播放

## 🔐 密码安全

### SHA256 加密

**密码存储方式:**
```
用户输入：123456
    ↓ SHA256
存储：8d969eef6ecad3c29a3a629280e686cf0c3f5d5a86aff3ca12020c923adc6c92
```

**安全特性:**
- ✅ 密码不以明文存储
- ✅ 使用行业标准哈希算法
- ✅ 即使数据库泄露也无法还原密码
- ✅ 每次哈希结果相同（确定性）

### 密码要求

| 要求 | 说明 |
|------|------|
| 最小长度 | 4 个字符 |
| 最大长度 | 无限制 |
| 字符类型 | 任意字符 |
| 大小写 | 敏感 |

### 密码建议

**强密码特征:**
- ✅ 至少 8 个字符
- ✅ 包含大小写字母
- ✅ 包含数字
- ✅ 包含特殊字符

**示例:**
```
弱密码：1234
中等：abc123
强密码：MyV1d30_2026!
```

- 自动删除最早记录
- 仅记录最近访问

## 🔌 API 接口

### 基础操作

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/password/<filename>` | GET | 检查密码状态 |
| `/api/password/<filename>` | POST | 验证密码 |
| `/api/password/<filename>` | PUT | 设置密码 |

### 高级功能

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/password/<filename>/hint` | GET | 获取密码提示 |
| `/api/password/<filename>/log` | GET | 获取访问日志 |
| `/api/password/<filename>/stats` | GET | 获取密码统计 |
| `/api/password/batch` | POST | 批量设置密码 |
| `/api/password/batch/remove` | POST | 批量移除密码 |
| `/api/password/protected` | GET | 获取受保护列表 |

### 使用示例

**设置密码:**
```javascript
fetch('/api/password/video.mp4', {
    method: 'PUT',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
        password: 'mypassword',
        hint: '我的密码'
    })
})
.then(r => r.json())
.then(data => {
    if (data.success) {
        console.log('密码已设置');
    }
});
```

**验证密码:**
```javascript
fetch('/api/password/video.mp4', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ password: 'mypassword' })
})
.then(r => {
    if (r.ok) {
        console.log('密码正确，可以播放');
    } else {
        console.log('密码错误');
    }
});
```

**批量设置:**
```javascript
fetch('/api/password/batch', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
        filenames: ['video1.mp4', 'video2.mp4'],
        password: 'commonpass',
        hint: '通用密码'
    })
})
.then(r => r.json())
.then(data => {
    console.log(`已为 ${data.count} 个视频设置密码`);
});
```

## 🐛 常见问题

### Q: 忘记密码怎么办？
A: 密码以哈希形式存储，无法找回。只能移除密码保护（需要访问设置）。

### Q: 密码区分大小写吗？
A: 是的，`Password` 和 `password` 是不同的密码。

### Q: 可以设置空密码吗？
A: 不可以，密码至少 4 个字符。

### Q: 访问日志保存多久？
A: 最多保存 100 条记录，超出自动删除最早的。

### Q: 批量操作有数量限制吗？
A: 没有硬性限制，但建议每次不超过 100 个视频。

### Q: 密码保护的视频能分享吗？
A: 可以分享链接，但对方需要密码才能观看。
