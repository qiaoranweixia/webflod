# 🎬 Flask 视频中心 - 专业版

一个功能完整的本地视频网站，支持视频上传、自动压缩、在线播放、播放列表管理等。

## ✨ 核心功能

### 📤 视频上传
- ✅ 多文件同时上传
- ✅ 实时进度显示
- ✅ **自动压缩**（节省 50-80% 空间）
- ✅ 拖拽上传支持
- ✅ 上传前预览（文件大小、时长）

### 🗜️ 智能压缩
- 使用 H.264 编码
- CRF 质量可控（默认 28）
- 自动缩放（最大 1280px 宽度）
- AAC 音频编码（128kbps）
- 压缩率显示

### 🎥 在线播放
- HTML5 播放器
- 支持主流视频格式
- 播放列表侧边栏
- 自动连播/循环
- 后台悬浮播放
- 📊 播放次数统计

### 📊 视频管理
- 缩略图自动生成
- 🎬 **悬停预览 GIF**（鼠标悬停自动播放 3 秒预览）
- 详细视频信息（分辨率、编码、时长）
- 🔍 搜索过滤
- 📄 分页浏览
- 🗑️ 批量删除
- ⭐ 收藏功能
- 📥 下载视频
- 📊 多种排序（日期/名称/大小/时长）
- 🌓 深色/浅色主题切换
- 🏷️ **视频分类/标签管理**

### 📱 响应式设计
- 完美适配手机/平板/桌面
- 触摸友好按钮
- 深色主题
- 流畅动画

## 📦 支持格式

**上传格式：** `mp4`, `webm`, `ogg`, `mov`, `avi`, `mkv`, `flv`, `wmv`

**输出格式：** `mp4` (H.264 + AAC)

## ⚙️ 配置选项

编辑 `config.py` 自定义设置：

```python
# 压缩配置
COMPRESS_VIDEO = True          # 是否自动压缩
COMPRESSION_PRESET = 'medium'  # 压缩速度：ultrafast ~ veryslow
CRF_VALUE = 28                 # 质量 (0-51, 越大越压缩)
MAX_WIDTH = 1280               # 最大宽度（保持比例）
AUDIO_BITRATE = '128k'         # 音频比特率

# 缩略图配置
GENERATE_THUMBNAIL = True      # 是否生成缩略图
THUMBNAIL_TIME = 5             # 在第几秒截图

# 服务配置
PORT = 5001                    # 服务端口
VIDEOS_PER_PAGE = 12           # 每页显示数量
```

### 压缩质量参考

| CRF 值 | 质量 | 文件大小 | 适用场景 |
|--------|------|----------|----------|
| 18-22  | 极高 | 较大 | 收藏/高质量 |
| 23-28  | 高   | 中等 | **日常使用（推荐）** |
| 29-35  | 中   | 较小 | 节省空间 |
| 36+    | 低   | 最小 | 极限压缩 |

## 🛠️ 高级功能

### 关闭自动压缩

上传时取消勾选"自动压缩"选项，或修改配置：

```python

```

### 批量上传

支持一次选择多个视频文件，自动排队上传。

### ⭐ 收藏功能

- 点击视频卡片上的"☆ 收藏"按钮添加到收藏

- 刷新页面后收藏状态依然保留

- 点击视频卡片上的"📥 下载"按钮直接下载
- 或在视频详情模态框中点击下载

### 🗑️ 批量删除

- 或点击"全选"一次性选择所有视频

- 帧率：2fps（平衡流畅度和文件大小）
- 宽度：320px（保持原始比例）
- 时长：3 秒（或视频剩余时长）
- 存储在 `thumbnails/` 目录

## 🐛 常见问题

### Q: 压缩很慢怎么办？
A: 在 `config.py` 中修改 `COMPRESSION_PRESET = 'fast'` 或 `'ultrafast'`

### Q: 压缩后画质太差？
A: 降低 `CRF_VALUE`（如改为 23），画质会更好但文件更大

### Q: 不支持某些格式？
A: 在 `config.py` 的 `ALLOWED_EXTENSIONS` 中添加扩展名

### Q: 如何备份视频？
A: 直接备份 `uploads/` 文件夹即可
