# 📋 播放列表/合集 - 完整功能指南

## 🎯 功能总览

播放列表功能允许你将视频组织成自定义合集，方便管理和分享。

### 核心功能

| 功能 | 说明 | 状态 |
|------|------|------|
| ➕ 创建播放列表 | 自定义名称和描述 | ✅ |
| 📹 添加视频 | 从视频卡片一键添加 | ✅ |
| 🗑️ 移除视频 | 从播放列表删除 | ✅ |
| ▶️ 连续播放 | 一键播放全部视频 | ✅ |
| 🔗 分享链接 | 生成分享 URL | ✅ |
| 📤 导出 | 导出为 JSON 文件 | ✅ |
| 📥 导入 | 从 JSON 文件导入 | ✅ |
| 🖼️ 自定义封面 | 设置播放列表封面 | ✅ |
| 🔢 重排序 | 拖拽调整视频顺序 | ✅ |
| 🔒 隐私设置 | 公开/私密播放列表 | ✅ |

## 🚀 使用指南

### 创建播放列表

**方法 1: 从播放列表管理**
1. 点击顶部导航栏的 📋 按钮
2. 点击"➕ 新建播放列表"
3. 输入名称和描述
4. 点击确认

**方法 2: 快速创建**
1. 点击视频卡片的"📋 合集"按钮
2. 选择"新建播放列表"
3. 输入名称
4. 视频自动添加

### 添加视频到播放列表

**从视频卡片:**
1. 找到想要添加的视频
2. 点击"📋 合集"按钮
3. 选择现有播放列表或创建新的
4. 完成！

**从播放列表管理:**
1. 打开播放列表管理 (📋)
2. 点击播放列表的"➕ 添加视频"
3. 输入视频文件名
4. 确认添加

### 播放播放列表

1. 打开播放列表管理 (📋)
2. 找到想要播放的列表
3. 点击"▶️ 播放全部"
4. 自动连续播放所有视频

### 管理播放列表

**移除视频:**
1. 打开播放列表管理
2. 找到要管理的播放列表
3. 点击视频标签上的 ✕

**删除播放列表:**
1. 打开播放列表管理
2. 点击播放列表右上角的 🗑️
3. 确认删除

**重排序:**
1. 打开播放列表
2. 拖拽视频调整顺序
3. 自动保存

## 🔗 分享功能

### 生成分享链接

1. 打开播放列表管理 (📋)
2. 找到要分享的播放列表
3. 点击"🔗 分享"按钮
4. 链接自动复制到剪贴板

### 分享链接格式

```
http://你的 IP:5001/playlist/import/<分享码>
```

### 导入他人分享的播放列表

**方法 1: 点击分享链接**
1. 点击他人分享的链接
2. 查看播放列表信息
3. 点击"✅ 导入到我的列表"
4. 完成！

**方法 2: 导入 JSON 文件**
1. 打开播放列表管理
2. 点击"📥 导入播放列表"
3. 选择 `.json` 文件
4. 自动导入

---

## 📤 导出/导入

### 导出播放列表

1. 打开播放列表管理
2. 点击要导出的播放列表的"📤 导出"
3. 自动下载 `.json` 文件
4. 文件包含所有播放列表信息

### 导入播放列表

1. 打开播放列表管理
2. 点击"📥 导入播放列表"
3. 选择之前导出的 `.json` 文件
4. 自动导入并显示

### 导出文件格式

```json
{
  "id": "dc757195",
  "name": "我的最爱",
  "description": "精选视频合集",
  "videos": [
    "video1.mp4",
    "video2.mp4",
    "video3.mp4"
  ],
  "cover": "",
  "is_public": true,
  "play_count": 0,
  "created_at": "2026-03-25 19:05:58",
  "updated_at": "2026-03-25 19:05:58"
}
```

## 🖼️ 自定义封面

### 设置播放列表封面

1. 打开播放列表管理
2. 点击播放列表的"⚙️ 设置"
3. 选择"设置封面"
4. 从现有缩略图选择或上传新图片
5. 保存

### 封面推荐尺寸

- **推荐**: 1280x720 (16:9)
- **最小**: 640x360
- **格式**: JPG, PNG
- **大小**: < 2MB

- ✅ 可生成分享链接

- 🔒 无法分享

1. 打开播放列表管理

3. 切换"公开/私密"开关
4. 保存

## 🔌 API 接口

### 播放列表管理

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/playlists` | GET | 获取所有播放列表 |
| `/api/playlists` | POST | 创建播放列表 |
| `/api/playlists/<id>` | GET | 获取单个播放列表 |
| `/api/playlists/<id>` | POST | 添加视频 |
| `/api/playlists/<id>/<filename>` | DELETE | 移除视频 |
| `/api/playlists/<id>` | DELETE | 删除播放列表 |

### 高级功能

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/playlists/<id>/reorder` | POST | 重排序 |
| `/api/playlists/<id>/cover` | PUT | 设置封面 |
| `/api/playlists/<id>/export` | GET | 导出 |
| `/api/playlists/import` | POST | 导入 |
| `/api/playlists/<id>/share` | GET | 生成分享链接 |
| `/playlist/import/<data>` | GET | 导入页面 |

## 🐛 常见问题

### Q: 播放列表能有多少个视频？
A: 没有硬性限制，但建议每个播放列表不超过 100 个视频以保持流畅。

### Q: 删除播放列表会删除视频吗？
A: 不会，只删除播放列表本身，视频文件不受影响。

### Q: 分享链接有效期多久？
A: 永久有效，除非原播放列表被删除。

### Q: 能否合并两个播放列表？
A: 目前不支持，但可以导出后手动合并 JSON 文件再导入。

### Q: 播放列表能设置密码吗？
A: 暂不支持，计划在未来版本添加。
