# 🎬 Flask 视频网站 - 功能更新日志

## 2026-03-25 v2.0 重大更新

### ✨ 新增功能

#### 📝 视频描述/备注系统
- **公开描述**: 支持 Markdown 格式的视频描述
  - 标题、列表、引用、代码块
  - 粗体、斜体、链接
  - 实时预览功能
  - 描述指示器（有描述的视频显示 📝 标记）
  
- **私人备注**: 仅自己可见的私密笔记
  - 记录拍摄时间、地点、人物
  - 适合不想公开的细节
  - 与公开描述分开存储

- **视频信息集成**: 详情页显示描述预览
  - 快速获取视频概要
  - 一键跳转到完整描述

#### 🏷️ 视频分类/标签系统
- **标签管理**: 添加、删除、编辑视频标签
- **常用标签**: 10 个预设标签快速选择
- **标签筛选**: 按标签过滤视频列表
- **标签统计**: 显示每个标签的视频数量
- **持久存储**: 标签数据保存在 `video_tags.json`

#### 🎬 悬停预览 (GIF)
- **自动预览**: 鼠标悬停播放 3 秒 GIF
- **智能生成**: 上传时后台异步生成
- **防闪烁**: 150ms 延迟避免快速移动闪烁
- **优化存储**: 2fps, 320px 宽度，平衡质量与大小

#### 🌓 主题切换
- **深色/浅色**: 一键切换界面主题
- **本地存储**: 主题偏好自动保存
- **平滑过渡**: CSS 动画过渡效果

#### 📥 下载功能
- **直接下载**: 点击按钮立即下载
- **详情下载**: 信息模态框也提供下载

#### ⭐ 收藏功能
- **本地收藏**: 浏览器 LocalStorage 存储
- **收藏筛选**: 一键显示所有收藏视频
- **持久化**: 刷新页面后状态保留

#### 🗑️ 批量操作
- **多选删除**: 勾选多个视频批量删除
- **全选功能**: 一键选择/取消所有视频
- **操作栏**: 选择后自动显示批量操作栏

#### 📊 排序选项
- **按日期**: 最新上传优先（默认）
- **按名称**: 字母顺序排列
- **按大小**: 文件大小排序
- **按时长**: 视频时长排序

#### 📈 播放统计
- **播放计数**: 自动记录每个视频播放次数
- **持久存储**: 数据保存在 `play_counts.json`

### 🎯 API 接口扩展

#### 标签管理
- `GET /api/tags` - 获取所有标签
- `GET /api/tags/<filename>` - 获取视频标签
- `POST /api/tags/<filename>` - 添加标签
- `DELETE /api/tags/<filename>` - 删除标签

#### 描述/备注
- `GET /api/description/<filename>` - 获取描述
- `POST /api/description/<filename>` - 保存描述
- `DELETE /api/description/<filename>` - 删除描述

#### 播放统计
- `POST /api/play/<filename>` - 记录播放

#### 预览
- `GET /previews/<filename>.gif` - 获取预览 GIF

### 🚀 使用建议

1. **首次使用**:
   - 上传视频后等待 GIF 生成（后台异步）
   - 添加标签分类便于后续查找
   - 为重要视频添加描述说明

2. **标签最佳实践**:
   - 使用一致的命名（如"旅行"而非"旅游"）
   - 一个视频可添加多个标签
   - 常用标签会自动出现在顶部

3. **描述模板**:
   ```markdown
   # 视频标题
   
   ## 拍摄信息
   - 时间：YYYY-MM-DD
   - 地点：xxx
   - 设备：xxx
   
   ## 亮点
   - 亮点 1
   - 亮点 2
   
   ## 备注
   其他需要说明的信息
   ```

4. **私人备注用途**:
   - 记录人物姓名
   - 拍摄参数设置
   - 后续拍摄计划
   - 敏感信息（不会公开）
