# 🖼️ 自定义封面 - 完整功能指南

## 🎯 功能总览

自定义封面功能允许你为视频上传专属封面图片或从视频关键帧中选择。

### 核心功能

| 功能 | 说明 | 状态 |
|------|------|------|
| 🎬 关键帧选择 | 从视频自动生成 3 个时间点封面 | ✅ |
| 📁 上传封面 | 上传自定义图片作为封面 | ✅ |
| 👁️ 实时预览 | 选择时即时预览效果 | ✅ |
| 🗑️ 移除封面 | 恢复为自动生成缩略图 | ✅ |
| 🖼️ 封面图库 | 浏览所有自定义封面 | ✅ |
| 🔒 格式验证 | 仅支持 JPG/PNG/WEBP | ✅ |
| 📦 大小限制 | 最大 5MB | ✅ |
| 📱 触摸优化 | 移动端友好界面 | ✅ |

## 🚀 使用指南

### 方法 1: 从视频关键帧选择

**推荐！快速且无需额外图片**

1. 找到想要设置封面的视频
2. 点击视频卡片的"⚙️ 设置"按钮
3. 点击"🎬 从视频选择"
4. 系统自动生成 3 个时间点的封面:
   - 第 1 帧：视频 10% 位置
   - 第 2 帧：视频 30% 位置
   - 第 3 帧：视频 50% 位置
5. 点击喜欢的封面
6. ✅ 自动应用！

### 方法 2: 上传自定义封面

**适合有专属图片的场景**

1. 找到想要设置封面的视频
2. 点击视频卡片的"⚙️ 设置"按钮
3. 点击"📁 上传自定义图片"
4. 选择图片文件
5. 等待上传完成
6. ✅ 封面已更新！

### 移除自定义封面

1. 点击视频卡片的"⚙️ 设置"按钮
2. 点击"🗑️ 移除封面"
3. 确认移除
4. ✅ 恢复为默认缩略图

## 🎬 关键帧选择

### 自动生成规则

系统从视频的 3 个时间点自动生成封面：

| 帧 | 位置 | 适用场景 |
|----|------|----------|
| 第 1 帧 | 10% | 开场画面 |
| 第 2 帧 | 30% | 早期内容 |
| 第 3 帧 | 50% | 中间内容 |

### 为什么选择这些时间点？

- **10%**: 避开片头黑屏，展示正片开始
- **30%**: 内容已展开，画面通常较好
- **50%**: 视频中间，代表性画面

### 示例

```
视频时长：10 分钟

自动生成封面：
- 第 1 帧：1:00 (10% 位置)
- 第 2 帧：3:00 (30% 位置)
- 第 3 帧：5:00 (50% 位置)
```

## 📁 上传自定义封面

### 支持的格式

| 格式 | 扩展名 | 推荐度 |
|------|--------|--------|
| JPEG | .jpg, .jpeg | ⭐⭐⭐⭐⭐ 推荐 |
| PNG | .png | ⭐⭐⭐⭐ |
| WebP | .webp | ⭐⭐⭐⭐ |

### 文件要求

- **最大尺寸**: 5MB
- **推荐分辨率**: 1280x720 (16:9)
- **最小分辨率**: 640x360
- **宽高比**: 16:9 (推荐)

### 上传步骤

1. 准备封面图片
2. 点击"📁 上传自定义图片"
3. 选择文件
4. 自动验证格式和大小
5. 上传成功提示

### 上传失败原因

| 错误 | 原因 | 解决方法 |
|------|------|----------|
| 文件过大 | > 5MB | 压缩图片 |
| 格式不支持 | 非 JPG/PNG/WEBP | 转换格式 |
| 文件损坏 | 图片无法读取 | 重新保存 |

## 🔌 API 接口

### 基础操作

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/cover/<filename>` | GET | 获取封面信息 |
| `/api/cover/<filename>` | PUT | 设置封面 |
| `/api/cover/<filename>` | DELETE | 删除封面 |
| `/covers/<filename>` | GET | 获取封面图片 |

### 高级功能

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/cover/upload/<filename>` | POST | 上传封面 |
| `/api/cover/select/<filename>` | GET | 获取封面选项 |
| `/api/cover/apply/<filename>` | POST | 应用选定封面 |
| `/api/covers/gallery` | GET | 获取封面图库 |

### 使用示例

**上传封面:**
```javascript
const file = input.files[0];
const formData = new FormData();
formData.append('cover', file);

fetch('/api/cover/upload/video.mp4', {
    method: 'POST',
    body: formData
})
.then(r => r.json())
.then(data => {
    if (data.success) {
        console.log('封面已更新');
    }
});
```

**获取封面选项:**
```javascript
fetch('/api/cover/select/video.mp4')
    .then(r => r.json())
    .then(data => {
        data.options.forEach(opt => {
            console.log(`封面 ${opt.time}s: ${opt.filename}`);
        });
    });
```

**应用封面:**
```javascript
fetch('/api/cover/apply/video.mp4', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ cover: 'custom_cover.jpg' })
})
.then(r => r.json())
.then(data => {
    if (data.success) {
        console.log('封面已应用');
    }
});
```

## 🐛 常见问题

### Q: 为什么关键帧生成失败？
A: 视频可能损坏或格式不支持，尝试上传自定义封面。

### Q: 上传的封面为什么显示模糊？
A: 图片分辨率过低，建议使用 1280x720 或更高分辨率。

### Q: 可以批量设置封面吗？
A: 目前不支持，需要逐个视频设置。

### Q: 封面文件存储在哪里？
A: 存储在 `thumbnails/` 目录，数据记录在 `custom_covers.json`。

### Q: 移除封面后原文件会删除吗？
A: 不会，文件保留在目录中，可重新应用。
