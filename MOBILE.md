## 📲 添加到主屏幕

### iOS Safari

1. 打开网站 `http://你的 IP:5001`
2. 点击底部 **分享** 按钮（⬆️ 方框箭头）
3. 滑动找到并点击 **"添加到主屏幕"**
4. 确认名称，点击右上角 **"添加"**

### Android Chrome

1. 打开网站 `http://你的 IP:5001`
2. 点击右上角 **菜单**（⋮）
3. 点击 **"添加到主屏幕"**
4. 确认名称，点击 **"添加"**

### Android Firefox

1. 打开网站
2. 点击右下角 **菜单**（⋮）
3. 点击 **"页面"**
4. 选择 **"添加到主屏幕"**

## 📴 离线使用

### 缓存策略

| 资源类型 | 策略 | 说明 |
|----------|------|------|
| HTML 页面 | 网络优先 | 失败时返回缓存 |
| CSS/JS | 缓存优先 | 后台更新 |
| 图片 | 缓存优先 | 失败时占位图 |
| 视频 | 仅在线 | 不缓存（太大） |
| API | 网络优先 | 失败返回离线提示 |

### 离线功能

**可用:**
- ✅ 浏览已访问页面
- ✅ 查看缓存的缩略图
- ✅ 访问设置和统计
- ✅ 查看本地收藏

**不可用:**
- ❌ 播放视频
- ❌ 上传新视频
- ❌ 获取最新数据
- ❌ 实时搜索

### 离线页面

当检测到网络断开时，自动显示离线页面：

```
┌─────────────────────────┐
│        📡               │
│     离线状态            │
│                         │
│  网络状态：❌ 离线      │
│  缓存模式：📦 只读      │
│  最后同步：xxx          │
│                         │
│  [🔄 重试连接]          │
│                         │
│  💡 提示                │
│  • 已缓存页面可浏览     │
│  • 视频播放需要网络     │
│  • 检查 Wi-Fi 或数据    │
└─────────────────────────┘
```

## ⚡ 性能优化

### 加载优化

1. **懒加载**: 图片和视频进入视口才加载
2. **资源压缩**: CSS/JS 自动压缩
3. **缓存策略**: 智能预加载关键资源
4. **流式传输**: 视频边下边播

### 渲染优化

1. **减少重绘**: CSS 动画使用 transform
2. **硬件加速**: 视频播放使用 GPU
3. **虚拟滚动**: 长列表优化（待实现）
4. **防抖节流**: 滚动和搜索优化

### 内存优化

1. **及时释放**: 关闭播放器清理资源
2. **限制缓存**: 控制缓存大小
3. **图片优化**: 适当压缩缩略图
4. **GC 友好**: 避免内存泄漏

## 🎯 响应式断点

```css
/* 超大屏幕 */
@media (min-width: 1400px) { }

/* 桌面 */
@media (min-width: 1024px) { }

/* 平板横屏 */
@media (max-width: 1024px) { }

/* 平板竖屏 */
@media (max-width: 768px) { }

/* 手机横屏 */
@media (max-width: 896px) and (orientation: landscape) { }

/* 手机竖屏 */
@media (max-width: 480px) { }

/* 触摸设备 */
@media (hover: none) and (pointer: coarse) { }

/* 减少动画 */
@media (prefers-reduced-motion: reduce) { }
```
